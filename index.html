HOW IT WORKS

MAD Smart Contracts

Mutually Assured Destruction — a game-theory mechanism where both parties lock deposits. Fraud means everyone loses, so honesty becomes the only rational choice.

Selling Crypto for FIAT

1
Seller creates MAD contract
Locks crypto value + equal security deposit (2× total)
2
Contract code shared
Seller sends the unique contract code to the buyer
3
Buyer accepts & deposits
Buyer locks an equal security deposit — both sides now have skin in the game
4
Buyer sends FIAT payment
Cash, bank transfer, or any agreed method — off-chain
5
Both confirm release
Each party confirms independently. Deposits return, crypto transfers to buyer
⚠
Cheat = Burn
If either party refuses to confirm, ALL locked funds are destroyed. No one profits from fraud.

Buying Crypto with FIAT

1
Request contract code
Ask the crypto seller for the MAD contract code
2
Accept contract in wallet
Paste the code, review the value, and lock your security deposit
3
Send payment to seller
Transfer the agreed FIAT amount (equal to deposit value) to the seller
4
Confirm release on your end
Signal that you've sent payment and are ready to finalize
5
Seller confirms receipt
Once the seller verifies your payment, they also confirm release
6
Crypto + deposits released
You receive the crypto, both security deposits return to their owners

Buying Physical Goods from Strangers

1
Buyer creates MAD contract
Locks payment value + equal security deposit in h173k
2
Seller accepts & deposits
Seller locks the same amount as security deposit
3
Seller ships the goods
Physical product, digital goods, services — anything
4
Buyer receives & verifies
Inspects the goods to ensure they match the agreement
5
Both confirm release
Deposits return, payment transfers to the seller
🛡
Mutual protection
Neither party can scam the other — both lose equally if the deal fails.
